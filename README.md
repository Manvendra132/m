
### Prerequisites:

- Node.js 8.x or above
- NPM

### How to Build this app locally

[Fork this repo](https://github.com/Manvendra132/m/fork) and then clone it:

```
git clone https://github.com/Manvendra132/m
```

`cd talk` and then install dependencies

```
npm install
```

Run the app

```
npm start
```
